# Riddle Master - Online Quiz Game

A premium, interactive riddle quiz application built with **Angular (v17+)** and **Tailwind CSS**. This project demonstrates modern Angular features like **Signals**, **Standalone Components**, and **Control Flow Syntax** (@if, @for), wrapped in a beautiful, responsive user interface.
